# thyroid-disease-classification
# Random Forest ile Tiroid Hastalığını Sınıflandırma

Bu projede UCI Machine Learning Repository üzerinde yer alan Thyroid0387 veri seti kullanılarak tiroid hastalığı sınıflandırması yapılmıştır.

## Kullanılan Yöntemler

- Random Forest
- Decision Tree
- Tuned Random Forest

## Değerlendirme Metrikleri

- Accuracy
- Sensitivity / Recall
- Specificity
- Precision
- F1-score

## Random Forest Sonuçları

- Accuracy: 96.29%
- Sensitivity / Recall: 93.33%
- Specificity: 97.34%
- Precision: 92.56%
- F1-score: 92.95%

## Veri Seti

UCI Thyroid Disease Dataset:  
https://archive.ics.uci.edu/dataset/102/thyroid+disease
